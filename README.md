Network Traffic Monitor — это приложение для мониторинга сетевого трафика в реальном времени. Программа предоставляет визуализацию сетевой активности, статистику по протоколам и анализирует объем передаваемых данных.
Основные возможности:
1. Отображение статистики трафика в реальном времени
2. Группировка по протоколам (HTTP, HTTPS, FTP)
3. Визуализация данных с графиками
4. Простой и интуитивный графический интерфейс
5. Работает без прав администратора (в демо-режиме)
 Установка
# Способ 1: Быстрый старт (рекомендуется)
# 1. Скачайте проект
git clone https://github.com/ваш-репозиторий/network-monitor.git
cd network-monitor

# 2. Установите зависимости
pip install matplotlib

# 3. Запустите программу
python network_monitor.py

# Способ 2: Установка с виртуальным окружением
# Создайте виртуальное окружение
python -m venv venv

# Активируйте окружение
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Установите зависимости
pip install matplotlib

# Запустите программу
python network_monitor.py
